# Esports Ethr Registry

## Introduction

Local ethr network using Truffle.

## How to use

```bash
# prerequisites
npm install -g truffle

# 1. move to directory
cd registry

# 2. run network with network name
truffle develop --network goerli

# 3. migrate (run in truffle shell)
> migrate
```

## How to test

```bash
# 1. move to test directory
cd test

# 2. install
pnpm i

# 3. run test
pnpm test

```
